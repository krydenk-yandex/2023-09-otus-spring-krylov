### Spring Framework Diploma Work
1. ```bash
    cp app.key users/src/main/resources/ \
    && cp app.pub users/src/main/resources/
   ```
2. ```bash
   docker-compose build
   ```
3. ```bash
   docker-compose up
   ```
4. Go to http://localhost:8080 and log in
5. **Credentials:**
   6. **a:1** - for admin user
   7. **u:1** - for common user
